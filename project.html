Web ApplicationPhotobooth User Album

hosted on AWS amplify

A proto-type read-only web application for a photobooth company to host client photos taken at events for view/download.

The frontend of this project uses React.js and the back-end is hosted on AWS. The code is pushed to CodeCommit where the code is automatically build, verified, and deployed to AWS Amplify.

Authenitcation is handled by AWS Cognito to register, log in, reset password, and log out. Validation tokens are utilized for extra measure of security.

Once a user is created, the Federation Identity ID is sent to the website owner, via Lambda to SES, to create the private S3 folder for that account.

All images stored on S3 are isolated from other users and are individually downloadable, as per IAM permissions. The entire folder is also downloadable as a .zip file.
